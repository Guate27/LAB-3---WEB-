Intructions 

In this repository, you will find an interactive story where you must make various decisions to help the protagonist reach an ending that you deem appropriate.

Getting Started

To begin the story, follow these steps:

1. Copy the following link: http://ssh.chicharronconpelos.shop/julio241071/Laboratorio3/pages/inicio/

2. Paste the link into the search bar of your preferred browser (for example, Google).

3. After pasting the link into your browser's search bar, press the Enter key.
   - NOTE: This action will take you directly to the first page of the story.

4. Begin your own story
